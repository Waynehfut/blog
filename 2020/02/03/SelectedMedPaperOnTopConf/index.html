Selected Paper on Top conference
2020-02-03|Word count: 788|Reading time: 4 min

Recently, I am preparing the literature review for a paper which related to the medical image. I will sort out the last year(time) top conference papers to make me more clear insight, which include CVPR, ICCV, MICCAI, ISBI, EMBC. Notice: Papers were collected conditionally.

Outline

  • CVPR 2019
  • ICCV 2019
  • MICCAI 2019
  • ISBI 2019
  • EMBC 2019

Timeline

CVPR 2019

Homepage

Since CVPR is not focus on the medical image, I just find only a few related papers.

  1. Collaborative Learning of Semi-Supervised Segmentation and Classification for Medical Images [Paper]

  2. Elastic Boundary Projection for 3D Medical Image Segmentation [Paper]

  3. Data Augmentation Using Learned Transformations for One-Shot Medical Image Segmentation [Paper]

  4. Learning Active Contour Models for Medical Image Segmentation [Paper]

  5. Machine Vision Guided 3D Medical Image Compression for Efficient Transmission and Accurate Segmentation in the Clouds [Paper]

  6. X2CT-GAN: Reconstructing CT From Biplanar X-Rays With Generative Adversarial Networks [Paper]

ICCV 2019

Homepage

Not like the CVPR, ICCV got a plenty of papers on this topic.

  1. Learning With Unsure Data for Medical Image Diagnosis [Paper]

  2. Recursive Cascaded Networks for Unsupervised Medical Image Registration [Paper] [Github]

  3. Learning Fixed Points in Generative Adversarial Networks: From Image-to-Image Translation to Disease Detection and Localization [Paper] [Github]

  4. A Deep Step Pattern Representation for Multimodal Retinal Image Registration [Paper].

  5. HistoSegNet: Semantic Segmentation of Histological Tissue Type in Whole Slide Images [Paper].

  6. Semantic-Transferable Weakly-Supervised Endoscopic Lesions Segmentation [Paper].

  7. Unsupervised Microvascular Image Segmentation Using an Active Contours Mimicking Neural Network [Paper].

  8. Conditional Recurrent Flow: Conditional Generation of Longitudinal Samples With Applications to Neuroimaging [Paper].

  9. A Deep Cybersickness Predictor Based on Brain Signal Analysis for Virtual Reality Contents [Paper].

  10. An Alarm System for Segmentation Algorithm Based on Shape Model [Paper].

  11. Align, Attend and Locate: Chest X-Ray Diagnosis via Contrast Induced Attention Network With Limited Supervision [Paper].

  12. DUAL-GLOW: Conditional Flow-Based Generative Model for Modality Transfer [Paper].

  13. Multi-Stage Pathological Image Classification Using Semantic Segmentation [Paper].

  14. GLAMpoints: Greedily Learned Accurate Match Points [Paper].

  15. Hallucinating IDT Descriptors and I3D Optical Flow Features for Action Recognition With CNNs [Paper].

  16. Joint Acne Image Grading and Counting via Label Distribution Learning [Paper].

  17. CAMEL: A Weakly Supervised Learning Framework for Histopathology Image Segmentation [Paper].

  18. Dilated Convolutional Neural Networks for Sequential Manifold-Valued Data [Paper].

  19. Prior-Aware Neural Network for Partially-Supervised Multi-Organ Segmentation [Paper].

MICCAI 2019

Homepage

I did not select all the papers here, just only a few papers which I am interested.

  1. Multi-stage Prediction Networks for Data Harmonization [Paper]

  2. Triplet Feature Learning on Endoscopic Video Manifold for Online GastroIntestinal Image Retargeting [Paper]

  3. An Extremely Fast and Precise Convolutional Neural Network for Recognition and Localization of Cataract Surgical Tools [Paper]

  4. Generating Large Labeled Data Sets for Laparoscopic Image Processing Tasks Using Unpaired Image-to-Image Translation [Paper]

  5. Incorporating Temporal Prior from Motion Flow for Instrument Segmentation in Minimally Invasive Surgery Video [Paper]

  6. Hard Frame Detection and Online Mapping for Surgical Phase Recognition [Paper]

  7. Automated Surgical Activity Recognition with One Labeled Sequence [Paper]

  8. Using 3D Convolutional Neural Networks to Learn Spatiotemporal Features for Automatic Surgical Gesture Recognition in Video [Paper]

  9. Surgical Skill Assessment on In-Vivo Clinical Data via the Clearness of Operating Field [Paper]

  10. Gastric Cancer Detection from Endoscopic Images Using Synthesis by GAN [Paper]

  11. Combined Learning for Similar Tasks with Domain-Switching Networks [Paper]

  12. Two-Stream CNN with Loose Pair Training for Multi-modal AMD Categorization [Paper]

  13. Multi-view Learning with Feature Level Fusion for Cervical Dysplasia Diagnosis [Paper]

  14. Selective Feature Aggregation Network with Area-Boundary Constraints for Polyp Segmentation [Paper]

  15. Decompose-and-Integrate Learning for Multi-class Segmentation in Medical Images [Paper]

  16. Multi-task Attention-Based Semi-supervised Learning for Medical Image Segmentation [Paper]

ISBI 2019

Homepage

ISBI is hosted by IEEE, but not as well-known as MICCAI or EMBC, so I just picked up a litte paper here.

  1. Multi-anatomy localization in fetal echocardiography videos [Paper]

  2. Biphasic Model Of Lung Deformations For Video-Assisted Thoracoscopic Surgery (VATS) [Paper]

  3. Focusnet: An Attention-Based Fully Convolutional Network for Medical Image Segmentation [Paper]

  4. XAI-CBIR: Explainable AI System for Content based Retrieval of Video Frames from Minimally Invasive Surgery Videos [Paper]

EMBC 2019

EMBC hosted by IEEE, but more focus on the biomedical inforamtion, not medical image.

  1. U-NetPlus: A Modified Encoder-Decoder U-Net Architecture for Semantic and Instance Segmentation of Surgical Instruments from Laparoscopic Images [Paper].

  2. Using a Motion Capture System as Reference for Motion Tracking in Photoplethysmography Imaging. [Paper].
