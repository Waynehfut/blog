жңәеҷЁеӯҰд№ еҸҜи§ЈйҮҠжҖ§д№ӢSHAPеҖј SHAP Values

2019-03-23 | жҠҖжңҜ | Hits | 1.5k Words | 5 Minutes

жң¬зҜҮзҝ»иҜ‘иҮӘKaggleжңәеҷЁеӯҰд№ еҸҜи§ЈйҮҠжҖ§еҫ®е…¬ејҖиҜҫрҹҺ“пјҢжң¬зҜҮж—¶з¬¬еӣӣиҜҫж—¶пјҢдё»и®ІеҲ©з”ЁSHAP(SHapley Additive exPlanations)еҖјз”Ёд»ҘеңЁдёҖж¬ЎзӢ¬з«Ӣйў„жөӢдёӯеҲҶи§ЈжЁЎеһӢгҖӮ

Check source at Kaggle by Dan Becker, translate by waynehfut

з®Җд»Ӣ

дҪ е·Із»ҸзңӢеҲ°пјҲжҲ–дҪҝз”ЁдәҶпјүд»ҺжңәеҷЁеӯҰд№ дёӯжҸҗеүҚеёёи§„жҙһеҜҹзҡ„зӣёе…іжҠҖжңҜгҖӮдҪҶжҳҜеҰӮжһңдҪ жғіеңЁдёҖж¬ЎзӢ¬з«Ӣйў„жөӢдёӯеҲҶи§ЈжЁЎеһӢзҡ„иҜқиҜҘжҖҺд№ҲеҒҡе‘ўпјҹ

SHAPеҖј(ShapleyеҠ жі•и§ЈйҮҠзҡ„зј©еҶҷ)еҸҜд»ҘеҲҶи§Јйў„жөӢз»“жһңд»Ҙеұ•зӨәжҜҸдёӘеҖјзҡ„еҪұе“ҚгҖӮйӮЈд№Ҳд»–еҸҜд»Ҙз”ЁеңЁд»Җд№Ҳең°ж–№е‘ўпјҹ

  • дёҖдёӘжЁЎеһӢиҜҙй“¶иЎҢдёҚеә”еҪ“иҙ·ж¬ҫз»ҷжҹҗдёӘдәәпјҢеҗҢж—¶жі•еҫӢиҰҒжұӮй“¶иЎҢйңҖиҰҒжҜҸж¬Ўи§ЈйҮҠжӢ’з»қиҙ·ж¬ҫзҡ„зјҳз”ұж—¶гҖӮ
  • дёҖдёӘеҒҘеә·жңҚеҠЎжңәжһ„жғізЎ®е®ҡе“Әдәӣеӣ зҙ еҜјиҮҙжҜҸдёӘжӮЈиҖ…жӮЈжңүжҹҗз§Қз–ҫз—…зҡ„йЈҺйҷ©пјҢд»ҺиҖҢд»–д»¬еҸҜд»Ҙжңүй’ҲеҜ№жҖ§зҡ„еҒҘеә·е№Ійў„е№¶зӣҙжҺҘи§ЈеҶіжҺүиҝҷдәӣйЈҺйҷ©еӣ зҙ гҖӮ

дҪ е°ҶеңЁиҝҷиҠӮиҜҫзЁӢдёӯдҪҝз”ЁSHAPеҖјжқҘи§ЈйҮҠеҚ•дёӘйў„жөӢгҖӮеңЁдёӢдёҖиҠӮиҜҫдёӯпјҢдҪ иҝҳе°ҶзңӢеҲ°е®ғеҰӮдҪ•дёҺе…¶д»–жЁЎеһӢзә§зҡ„жҙһеҜҹж–№жі•з»“еҗҲгҖӮ

еҰӮдҪ•иҝҗдҪң

SHAPеҖји§ЈйҮҠдәҶеҜ№дәҺз»ҷе®ҡзү№еҫҒе…·жңүжҹҗдәӣзү№е®ҡеҖјжүҖдә§з”ҹзҡ„еҪұе“ҚпјҢе№¶дёҺжҲ‘д»¬еңЁиҜҘзү№еҫҒе…·жңүжҹҗдәӣеҹәзәҝеҖјж—¶жүҖдҪңзҡ„йў„жөӢиҝӣиЎҢжҜ”иҫғгҖӮ

дёҖдёӘзӨәдҫӢе°ҶжңүеҲ©дәҺзҗҶи§ЈпјҢжҲ‘д»¬жң¬иҠӮд»Қе°ҶдҪҝз”ЁеүҚж–ҮжҺ’еҲ—йҮҚиҰҒжҖ§е’ҢйғЁеҲҶдҫқиө–еӣҫжүҖдҪҝз”Ёзҡ„и¶ізҗғ/ж©„жҰ„зҗғзҡ„дҫӢеӯҗгҖӮ

еңЁд№ӢеүҚзҡ„иҜҫзЁӢдёӯпјҢжҲ‘д»¬йў„жөӢдәҶзҗғйҳҹжҳҜеҗҰдјҡжңүеҜ№дәҺиҺ·еҫ—жңҖдҪізҗғе‘ҳеҘ–гҖӮ

жҲ‘д»¬д№ҹи®ёдјҡй—®пјҡ

  • еҰӮжһңзҗғйҳҹжү“е…ҘдәҶ3зҗғдјҡжҖҺж ·жҺЁеҠЁйў„жөӢз»“жһңпјҹ

е…¶е®һиҝҷдёӘй—®йўҳеҸҜд»ҘжӣҙеҠ е…·дҪ“пјҢйҮҸеҢ–зҡ„еӣһзӯ”еҸҜд»Ҙиў«йҮҚеҶҷдёәпјҡ

  • еҰӮжһңзҗғйҳҹжү“е…ҘдәҶ3зҗғиҖҢдёҚжҳҜдёҖдәӣеҹәеҮҶиҝӣзҗғж•°иҝҷдёҖдәӢе®һе°ҶдјҡжҖҺж ·жҺЁеҠЁйў„жөӢз»“жһңпјҹ

еҪ“з„¶пјҢжҜҸдёӘйҳҹдјҚйғҪдјҡжңүеҫҲеӨҡзҡ„зү№еҫҒгҖӮеӣ жӯӨжҲ‘д»¬еҰӮжһңеӣһзӯ”иҝӣзҗғж•°иҝҷдёӘй—®йўҳпјҢжҲ‘д»¬д№ҹеҸҜд»ҘдёәжүҖжңүе…¶д»–еҠҹиғҪйҮҚеӨҚжӯӨиҝҮзЁӢгҖӮ

SHAPеҖје°Ҷд»ҘдҝқиҜҒиүҜеҘҪеұһжҖ§зҡ„ж–№ејҸжү§иЎҢжӯӨж“ҚдҪңгҖӮеҪ“жҲ‘д»¬иҝӣиЎҢеҰӮдёӢзҡ„йў„жөӢж—¶пјҡ sum(SHAP values for all features) = pred_for_team - pred_for_baseline_values д№ҹе°ұжҳҜиҜҙпјҢжүҖжңүзү№еҫҒзҡ„SHAPеҖјжҳҜеҜ№дёәд»Җд№Ҳйў„жөӢдёҺеҹәзәҝдёҚеҗҢзҡ„дёҖдёӘжұӮе’ҢгҖӮиҝҷд№ҹе…Ғи®ёжҲ‘д»¬е°Ҷйў„жөӢеҲҶи§ЈдёәдёӢеӣҫпјҡ

йӮЈеҰӮдҪ•зҗҶи§Јиҝҷдәӣе‘ўпјҹ

еҪ“жҲ‘д»¬дҪңеҮә0.7зҡ„йў„жөӢж—¶(еҚіиҜҘйҳҹжңү70%жҰӮзҺҮиҺ·еҫ—жңҖдҪізҗғе‘ҳпјҢwaynehfutжіЁ)пјҢдёҺжӯӨеҗҢж—¶еҹәзәҝеҖјжҳҜ0.4979гҖӮзү№еҫҒеҖјеҜјиҮҙйў„жөӢеҖјзҡ„еўһй•ҝзҡ„з”ұзІүиүІеҢәеҹҹж ҮеҮәпјҢиҖҢе®ғзҡ„и§Ҷи§үе°әеҜёеӨ§е°ҸиЎЎйҮҸдәҶзү№еҫҒзҡ„еҪұе“ҚгҖӮзү№еҫҒеҜјиҮҙзҡ„йҷҚдҪҺж•Ҳжһңз”ұи“қиүІеҢәеҹҹж ҮеҮәгҖӮеҪұе“ҚеҠӣжңҖеӨ§зҡ„еҖјжҳҜжқҘиҮӘдәҺGoal Scoredдёә2гҖӮдёҺжӯӨеҗҢж—¶жҺ§зҗғзҺҮе…·жңүйҷҚдҪҺйў„жөӢз»“жһңзҡ„ж•ҲжһңгҖӮ

еҰӮжһңд»ҺзІүзәўжқЎзҡ„й•ҝеәҰдёӯеҮҸеҺ»и“қжқЎзҡ„й•ҝеәҰпјҢеҲҷе®ғзӯүдәҺд»ҺеҹәеҖјеҲ°з»“жһңзҡ„и·қзҰ»гҖӮ

иҝҷйЎ№жҠҖжңҜд»ҚжңүдёҖдәӣеӨҚжқӮжҖ§пјҢдёәдәҶзЎ®дҝқеҹәзәҝеҠ дёҠеҗ„дёӘзӢ¬з«Ӣйў„жөӢзҡ„жҖ»е’ҢзӣёеҠ зҡ„еҖјдёҺйў„жөӢеҖјзӣёзӯү(еҗ¬дёҠеҺ»дјјд№ҺдёҚжҳҜйӮЈд№Ҳз®ҖеҚ•)гҖӮжҲ‘д»¬е°ҶдёҚдјҡеңЁиҝҷйҮҢи®Ёи®әз»ҶиҠӮпјҢеӣ дёәиҝҷдёӘжҠҖжңҜе№¶дёҚйҮҚиҰҒпјҢж¬ІзҹҘиҜҰжғ…еҸҜд»ҘеңЁиҝҷдёӘеҚҡж–Үдёӯеҫ—еҲ°и§ЈйҮҠ

и®Ўз®—SHAPеҖјзҡ„д»Јз Ғ

жҲ‘д»¬е°ҶдҪҝз”Ёдјҳз§Җзҡ„Shapеә“жқҘи®Ўз®—SHAPеҖјгҖӮ

дҫӢеҰӮпјҢжҲ‘д»¬е°ҶйҮҚз”ЁдҪ е·Із»ҸзңӢиҝҮжүҖжңүж•°жҚ®зҡ„жЁЎеһӢгҖӮ

жҲ‘д»¬е°ҶжҹҘзңӢж•°жҚ®йӣҶдёӯеҚ•еҲ—ж•°жҚ®зҡ„SHAPеҖјпјҲжҲ‘д»¬йҡҸжңәйҖүжӢ©з¬¬дә”иЎҢпјүгҖӮеҜ№дәҺе…¶дёӯеҶ…е®№пјҢжҲ‘д»¬еңЁжҹҘзңӢSHAPеҖјд№ӢеүҚе…ҲзңӢеҺҹе§Ӣйў„жөӢеҖјгҖӮ

array([[0.3, 0.7]])

иҝҷдёӘйҳҹдјҚжңү70%зҡ„жҰӮзҺҮиҺ·еҫ—иҝҷдёӘеҘ–йЎ№гҖӮ

жҺҘзқҖжҲ‘д»¬жқҘе…іжіЁдёӢеҚ•ж¬Ўйў„жөӢзҡ„SHAPеҖј

дёҠиҝ°зҡ„shap_valuesжҳҜдёҖдёӘжңүдёӨдёӘж•°з»„зҡ„еҲ—еҜ№иұЎгҖӮз¬¬дёҖдёӘж•°з»„SHAPеҖјиЎЁзӨәиҙҹйқўзҡ„иҫ“еҮәпјҲж— жі•иҺ·еҫ—еҘ–пјүпјҢSHAPеҖјзҡ„з¬¬дәҢдёӘж•°з»„жҳҜд»ЈиЎЁдәҶжӯЈйқўзҡ„иҫ“еҮәпјҲиөўеҫ—жҜ”иөӣпјүгҖӮжҲ‘д»¬йҖҡеёёдјҡиҖғиҷ‘йў„жөӢз§ҜжһҒз»“жһңзҡ„йў„жөӢпјҢеӣ жӯӨжҲ‘д»¬е°ҶжҠҠжүҖжңүSHAPеҖјзҡ„з§ҜжһҒиҫ“еҮәжҸҗеҮәпјҲдҪҝз”Ёshap_values[1]пјү

жҹҘзңӢеҺҹе§Ӣж•°з»„еҫҲйә»зғҰпјҢдҪҶжҳҜshapеҢ…жңүдёҖдёӘеҸҜи§ҶеҢ–з»“жһңзҡ„еҘҪж–№жі•гҖӮ

Outпјҡ

еҰӮжһңдҪ д»”з»Ҷзҡ„жҹҘзңӢеҲӣе»әSHAPеҖјзҡ„д»Јз ҒпјҢдҪ е°ҶдјҡжіЁж„ҸеҲ°жҲ‘д»¬еҸӮиҖғдәҶshap.TreeExplainer(my_model)дёӯзҡ„ж ‘гҖӮдҪҶжҳҜSHAPеҢ…е·Із»Ҹи§ЈйҮҠдәҶжЁЎеһӢзҡ„жҜҸз§Қзұ»еһӢгҖӮ

  • shap.DeepExplainerеңЁж·ұеәҰжЁЎеһӢдёӯжңүж•Ҳжһң
  • shap.KernelExplainerеңЁжүҖжңүжЁЎеһӢдёӯйғҪжңүж•ҲпјҢдҪҶд»–жҜ”е…¶д»–зҡ„и§ЈйҮҠеҷЁж…ўдәҶдёҖдәӣпјҢдё”д»–жҸҗдҫӣдәҶиҝ‘дјјеҖјиҖҢдёҚжҳҜеҮҶзЎ®еҖј

дёӢйқўжҸҗдҫӣдәҶдёҖдёӘдҪҝз”ЁKernelExplainerзҡ„дҫӢеӯҗд»ҘиҺ·еҸ–зӣёдјјзҡ„з»“жһңгҖӮз»“жһңеҖјдёҚжҳҜе®Ңе…ЁдёҖиҮҙпјҢеӣ дёәKernelExplainerз»ҷеҮәдәҶдёҖдёӘиҝ‘дјјзҡ„з»“жһңгҖӮдҪҶжҳҜз»“жһңеҖјиЎЁзӨәзҡ„ж„ҸжҖқжҳҜдёҖиҮҙзҡ„гҖӮ

жңәеҷЁеӯҰд№ 
жүӢжңҜи§Ҷйў‘ж•°жҚ®йӣҶж•ҙзҗҶжңәеҷЁеӯҰд№ еҸҜи§ЈйҮҠжҖ§д№ӢйғЁеҲҶдҫқиө–еӣҫ Partial Dependence Plots
